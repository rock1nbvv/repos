расчёт базовых арифметических выражений
расчет выражений со скобками(например: 1+(2*3-10/2) )
проведение расчетов с десятичными дробями(запись дробных чисел через точку".")

расчёты с базовыми тригонометрическими функциями sin, cos, tan(запись:sin(число), cos(число), tan(число))
расчёты выражений с базовыми логическими операциями: конъюнкция(/\), дизъюнкция(\/), отрицание(!)
расчёты с переменными. Переменные указываются до записи выражения через точку с запятой(например: a=1;b=-2)
